# Skin Lesion Classifier

A web application for classifying skin lesions as Benign or Melanoma using a trained ResNet18 deep learning model.

## Project Details

**Group Members:** AHMED AYASH, OMAR ADEL MOHAMMED, AYAH RAJOUB, GHADA FRIGUI

**Institution:** İstinye Üniversitesi

**Year:** 2026

## Features

- Upload dermoscopy images
- AI-powered prediction (Benign or Melanoma)
- Confidence scores
- Real-time results

## Model

- **Architecture:** ResNet18 with Transfer Learning
- **Dataset:** PH2 Dermoscopy Dataset
- **Framework:** PyTorch + Gradio

## Usage

Upload a dermoscopy image and get instant predictions!

⚠️ **Disclaimer:** For research purposes only, not a substitute for professional medical diagnosis.
